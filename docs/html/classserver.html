server::server ( )

Server class.

Server reads the XML file at start and creates camera controller classes with the data in the XML file. The server creates a camera controller object for each camera defined in the XML file.

Server is a Singleton.

void server::parsexmldata ( )

Server parse data from XML file to camera class objects.

The TinyXML-2 parser tuns through every line and takes the DEFINED (look up the XML) data to create camera class object. The parser uses the defined function 'getElementText'.
