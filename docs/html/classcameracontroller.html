cameracontroller::cameracontroller ( )

Cameracontroller class.

Simulates a controller for a camera. Each controller controls and simulates a camera defined in the XML. The client device doesn't communicate directly with the camera, it communicates with this camera class, which opens sockets with which the client device can communicate.
